Maximum Flow Problem - Dinitz’s Algorithm - Java

In this academic project, the objective was to solve the maximum flow problem. To achieve the Dinitz’s algorithm has been implemented using the Java language. A static class representing a graph with a maximum flow for each vertex is implemented within the program. Several functionalities are available in this program to modify the graph topology. For example, you have the possibility to delete, add, modify the maximum flow of a specific node. A method to display the actual state of the graph is also available. Moreover, to observe the asymptotic running time of this algorithm, a method providing a comparative analysis has been implemented. This method retrieves the time spent by the algorithm to solve the problem on datasets of different sizes.

Neural Network - C

In this personal project, the aim is to understand the core functioning of neural networks. Convolutional Neural Network for Times Series forecasting - Python

In this personal project, a Convolutional Neural Network (CNN) is implemented to forecast evapotranspiration data. The program preprocesses the data before feeding them to a model with a Conv1D layer, a MaxPooling1D, a Flatten and two Dense layers. The program uses ‘pyplot’ to plot the loss and the validation loss for each epoch of the training. And finally, plot the forecasted value with the actual value to have a visual representation of the performance of the model. This project has been the opportunity to learn a new way to use a deep learning method for time series forecasting.

Simple Encoding/Decoding script - Python

This academic project written in Python provide a very basic way to encrypt and decrypt a string. You have three functionalities, the first one is to encode a string by feeding the program with a string and an integer. The process follows the Caesar cipher and shift each letter by the specified integer. The second option provide an implementation do to the opposite. The user input an encoded string and an integer, and the decoded string is outputed. The last option provides an implemenation to find the shift number. The user input a string and a word from within the string, then, when the shifting integer is found, the string is decoded. This project allowed me to understand to very basic process behind encryption.
