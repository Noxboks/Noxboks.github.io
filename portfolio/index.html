Maximum Flow Problem - Dinitz’s Algorithm - Java

In this academic project, the objective was to solve the maximum flow problem. To achieve the Dinitz’s algorithm has been implemented using the Java language. A static class representing a graph with a maximum flow for each vertex is implemented within the program. Several functionalities are available in this program to modify the graph topology. For example, you have the possibility to delete, add, modify the maximum flow of a specific node. A method to display the actual state of the graph is also available. Moreover, to observe the asymptotic running time of this algorithm, a method providing a comparative analysis has been implemented. This method retrieves the time spent by the algorithm to solve the problem on datasets of different sizes.

Stacking Ensemble Model - Hyperparameters optimisation - Python

As part of the Final Year Project developed at the University of Westminster, a Stacking Ensemble Neural Network has been developed. We use the Sherpa library to tune the hyperparameters of the Ensemble model. The GPyOpt Bayesian algorithm has to be wrapped to be usable by the Sherpa library. The compiled neural network needs to run within a loop to allow the call-backs of the model to assess the validation loss after training. A list of Sherpa data structure items representing the grid that will be searched needs to be set. The Bayesian optimisation algorithm needs to incorporate the parameters and the algorithm itself. Finally, the Bayesian Algorithm needs to be fitted with the data to start optimising the model.

Convolutional Neural Network for Times Series forecasting - Python

In this personal project, a Convolutional Neural Network (CNN) is implemented to forecast evapotranspiration data. The program preprocesses the data before feeding them to a model with a Conv1D layer, a MaxPooling1D, a Flatten and two Dense layers. The program uses ‘pyplot’ to plot the loss and the validation loss for each epoch of the training. And finally, plot the forecasted value with the actual value to have a visual representation of the performance of the model. This project has been the opportunity to learn a new way to use a deep learning method for time series forecasting.

Final Year Project - C# / Python

This project focuses on analysing a specific area of environmental finance, the European Union Allowances (E.U.A.). It highlights the opportunity to investigate this particular commodity and design, develop, and deploy software to forecast the European Union Allowances spot price using a deep learning method known as ‘Ensemble Method for Time Series’. This project aims to reduce greenhouse gas emissions on mother nature by providing financial and energy experts with a tool to improve their trading decisions regarding the E.U.A. and indirectly attract more funds to low-emission projects. The project follows a strict procedure to accumulate knowledge on the academic research related to the European Union Trading System, the European Union Allowances and its impact, state of the art Deep Learning, Machine Learning and statistical treatment methods. The Dynamic Systems Development Method followed through this project and the different iterations to design, implement, and deploy this project are overviewed iteratively by industry experts to integrate the system within a real-world environment.
