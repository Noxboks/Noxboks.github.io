Maximum Flow Problem - Dinitz’s Algorithm - Java

In this academic project, the objective was to solve the maximum flow problem. To achieve the Dinitz’s algorithm has been implemented using the Java language. A static class representing a graph with a maximum flow for each vertex is implemented within the program. Several functionalities are available in this program to modify the graph topology. For example, you have the possibility to delete, add, modify the maximum flow of a specific node. A method to display the actual state of the graph is also available. Moreover, to observe the asymptotic running time of this algorithm, a method providing a comparative analysis has been implemented. This method retrieves the time spent by the algorithm to solve the problem on datasets of different sizes.

Neural Network - C

In this personal project, the aim is to understand the core functioning of neural networks. To achieve this, a very simple one has been implemented by implementing the activation functions, the shuffling functions to reduce the decay gradient descent as well as the topology of the artificial neural network. The computational methods processing the inputs and the activation for each layer, the changes for each layer and its application to adapt the different weights and biases have also been implemented during this project. Each step is implemented manually and not used from an API allow a proper understanding of the core structure and functioning of the neural networks heavily used nowadays. In addition, it has been the opportunity to code in the C language and start learning about the philosophy behind this language.

Convolutional Neural Network for Times Series forecasting - Python

In this personal project, a Convolutional Neural Network (CNN) is implemented to forecast evapotranspiration data. The program preprocesses the data before feeding them to a model with a Conv1D layer, a MaxPooling1D, a Flatten and two Dense layers. The program uses ‘pyplot’ to plot the loss and the validation loss for each epoch of the training. And finally, plot the forecasted value with the actual value to have a visual representation of the performance of the model. This project has been the opportunity to learn a new way to use a deep learning method for time series forecasting.

Academic Research Project

Many companies use both relational and non-relational databases to solve various problems. If there is a doubt that the database may become the bottleneck of a project based on working with large amounts of information, it is worth looking in the direction of NoSQL databases that allow something that relational databases are unable to realise. Therefore, this research project tries to verify which types of tasks are best to approach with a specific database and if we can simulate one type of database in another type and deduct if there is an advantage by doing this.

Final Year Project

This project focus on analysing a specific area of environmental finance, the European Union Allowances (EUA). It is an opportunity to investigate this particular commodity, as well as, design, develop and deploy software to forecast the European Union allowances Spot price. The very core of the software will be constituted by a deep learning model based on a method known as Ensemble Method for Time Series and its hyperparameter selected using Bayesian optimisation. This project aims to provide a tool for financial and/or energy experts to improve their trading decisions regarding the EUA.
